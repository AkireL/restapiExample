# Rest API whith Node.js

## Support of user

#### End points

| Verb | Route| Description|
|------|:------:|----------:|
|Get| "/"| Get all users|
|Post| "/"| Create user|
|Get| "/:id"| Get user by id|
|Put| "/:id"| Update user|
|Delete| "/:id"|Delete user|

User's data
- id
- userName
- name
- email
- password
- createDate
